# 1. プロジェクト概要

JSON文字列を整形・圧縮し、見やすく表示するWebツールを作成する。

開発者が日常的に利用できるツールを目指す。

---

# 2. 作成目的

- JavaScriptによるDOM操作の理解を深める
- JSONの理解を深める
- 実務で利用できるツールを制作する
- ポートフォリオとして公開する

---

# 3. ターゲットユーザー

- Webエンジニア
- プログラマー
- 学習者

---

# 4. 機能一覧

## 4.1 基本機能

- JSONを入力できる
- JSONを整形できる
- JSONを圧縮（Minify）できる
- 整形後のJSONを表示する

## 4.2 UI機能

- 入力エリアを表示する
- 出力エリアを表示する
- ボタンを表示する
- レスポンシブ対応

---

# 5. 使用技術

- HTML
- CSS
- JavaScript

---

# 6. 非機能要件

- レスポンシブ対応
- アクセシビリティ対応
- エラー時に分かりやすいメッセージを表示する
- 入力内容はブラウザ上のみで処理する
