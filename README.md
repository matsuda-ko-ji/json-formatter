# JSON Formatter

JSONを整形・圧縮するWebツールです。

## 概要

このアプリケーションは、入力したJSONを見やすく整形したり、
不要な空白を削除して圧縮したりできるWebツールです。

## 開発目的

JSONを扱う機会が多いWeb開発において、
日常的に利用できるツールを制作することを目的としました。

## 機能

- JSON整形
- JSON圧縮
- エラーメッセージ表示
- レスポンシブ対応

## 使用技術

- HTML
- CSS
- JavaScript
